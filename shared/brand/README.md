# Veiron brand assets

**Source of truth:** `veiron-desktop-tauri/logo.png` (1600×1600).

Generated variants in this folder (`logo-32` … `logo-512`, `logo-mark.png`) are used by:

| Consumer | Path |
|---|---|
| Desktop Tauri | `veiron-desktop-tauri/public/logo.png` |
| VPS admin panel | `veiron-release/vps-control-plane/admin-server/static/logo.png` |
| Android | `veiron-android/app/src/main/res/drawable/logo.png` + mipmaps |
| Installer icons | generated via `scripts/generate-icons.ps1` in Tauri |

Do not invent alternate logos. Re-run generation from the source PNG when the brand mark changes.
